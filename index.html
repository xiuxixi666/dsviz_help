四种数据结构的基本信息

链表（Linked List）

  • 定义：由一系列节点组成，每个节点包含数据和指向下一个节点的指针。
  • 特点：插入/删除操作灵活，随机访问效率较低。
  • 常见操作：插入节点、删除节点、查找节点。
  • 时间复杂度：按位置插入/删除 O(1)，按值查找 O(n)。
  • 常见类型：
    • 单链表：节点只包含一个指向下一个节点的指针。
    • 双向链表：每个节点包含指向前驱和后继的指针。
    • 循环链表：尾节点指向头节点，形成环。
  • 典型应用：动态内存分配、LRU 缓存、操作系统任务调度。

栈（Stack）

  • 定义：一种线性结构，只允许在一端进行插入和删除操作。
  • 特点：后进先出（LIFO，Last In First Out）。
  • 常见操作：push（入栈）、pop（出栈）、top/peek（读取栈顶元素）。
  • 时间复杂度：插入/删除/访问栈顶均为 O(1)。
  • 典型应用：函数调用栈、括号匹配、撤销操作、深度优先搜索（DFS）。

队列（Queue）

  • 定义：一种线性结构，元素只能在队尾插入，在队头删除。
  • 特点：先进先出（FIFO，First In First Out）。
  • 常见操作：enqueue（入队）、dequeue（出队）、front（读取队头元素）。
  • 时间复杂度：插入/删除均为 O(1)。
  • 变体：
    • 循环队列：空间利用率更高。
    • 双端队列（Deque）：允许两端插入和删除。
    • 优先队列（Priority Queue）：元素带优先级，出队顺序由优先级决定。
  • 典型应用：任务调度、消息缓冲、广度优先搜索（BFS）。

二叉树（Binary Tree / BST）

  • 定义：一种层次结构，每个节点最多有两个子节点（左子树和右子树）。
  • 遍历方式：
    • 前序遍历：根 → 左 → 右。
    • 中序遍历：左 → 根 → 右。
    • 后序遍历：左 → 右 → 根。
    • 层序遍历：按层逐层访问。
  • 二叉搜索树（BST）：中序遍历结果有序；查找、插入、删除平均时间复杂度 O(log n)。
  • 常见变体：
    • 平衡二叉树（AVL、红黑树）：保持树高度平衡，保证 O(log n) 操作。
    • 堆（Heap）：完全二叉树，常用于优先队列。
    • 字典树（Trie）：用于字符串快速检索。
  • 典型应用：快速查找与排序、表达式解析、索引构建、路径搜索。

使用手册

本工具主要用于 栈、队列、链表、二叉树 的可视化学习与演示。典型的操作流程如下：

  1. 新建页面：通过左侧按钮或标签页右键菜单，新建一种数据结构的演示页面。
  2. 执行操作：在工具栏或菜单中选择相应操作（如链表插入、栈入栈、队列入队、树插入节点）。操作结果会在中间画布动态展示。
  3. 观察信息：右下角的 DS Info 会显示元素数量等统计信息。刚进入页面时显示「——」，执行操作后显示实际计数。
  4. 保存与加载：通过 File → Save/Load 保存当前数据结构为 .dsf 文件，或从文件中加载之前的状态。
  5. 清空与重置：Operate → Clear 可以清空当前结构；Reset 可以恢复默认视图。
  6. 动画速度：通过 Operate → Speed 菜单选择演示速度（慢 / 中 / 快）。
  7. 帮助与反馈：菜单 Help → About 可查看本页面链接；如有问题可在留言区或问题反馈入口提出。

常见问题（FAQ）

保存/加载失败？确认扩展名为 .dsf 且有权限；文件内容需符合格式。

动画太快/太慢？通过菜单或快捷键 Alt+1/2/3 调整。

右下角显示 0/100？切换页刚进入会显示「——」，有操作后才显示实际计数。
