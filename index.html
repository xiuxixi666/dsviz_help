四种数据结构的基本信息

栈（Stack）

  • 特点：后进先出（LIFO）。
  • 常用操作：push 入栈、pop 出栈、top/peek 读栈顶。
  • 典型场景：撤销/回溯、括号匹配、DFS。

队列（Queue）

  • 特点：先进先出（FIFO）。
  • 常用操作：enqueue 入队、dequeue 出队、front 读队头。
  • 典型场景：消息缓冲、调度、BFS。

链表（Linked List）

  • 由节点通过指针相连；已知位置时插入/删除 O(1)，随机访问慢。
  • 常见变体：单链表、双向链表、循环链表。
  • 典型场景：频繁插入/删除的场合、LRU 缓存。

二叉树（Binary Tree / BST）

  • 每个节点最多两个子节点；遍历：前序/中序/后序/层序。
  • BST（搜索二叉树）：中序有序，查找/插入/删除平均 O(log n)。
  • 典型场景：有序数据组织、区间/优先级结构的基础。

使用手册

  • 新建页面：左侧快捷按钮或标签上右键菜单。
  • 保存/加载：File → Save/Load（文件后缀 .dsf）。
  • 清空/重置：Operate → Clear/Reset。
  • 动画速度：Operate → Speed（慢/中/快）。
  • DS Info：切换页面时显示「XX元素：——」，操作后显示真实计数。
  • About：帮助菜单中的 About 提供本页链接。

快捷键

  • Ctrl+S 保存　　Ctrl+O 加载
  • Ctrl+L 清空　　Ctrl+R 重置视图
  • Alt+1 / Alt+2 / Alt+3 切换速度（慢/中/快）
  • Esc 关闭当前标签页

常见问题（FAQ）

保存/加载失败？确认扩展名为 .dsf 且有权限；文件内容需符合格式。

动画太快/太慢？通过菜单或快捷键 Alt+1/2/3 调整。

右下角显示 0/100？切换页刚进入会显示「——」，有操作后才显示实际计数。
